# Terminology

## 보조 패널(side panel)

- Official Korean name: `보조 패널`
- Official bilingual name: `보조 패널(side panel)`
- CLI namespace: `side-panel`
- JSON and memory prefix: `side_panel`
- Documentation file prefix: `side-panel`

## Runtime Boundary

- Public repo: source code, tests, public docs, plugin metadata, reusable schemas, and synthetic fixtures.
- Private profile workspace: real lesson DB files, learner records, generated reports, logs, backups, and private memory.

## Deprecated Term Notes

Do not introduce new official names such as `right panel`, `우측패널`, `assistant panel`,
or `html panel`.
